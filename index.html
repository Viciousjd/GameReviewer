Reviewer Quiz

1. What is the first step in disassembling a computer system unit?





Correct Answer: a) Disconnecting all external peripherals

2. When assembling a computer, what should you do before installing the processor?





Correct Answer: b) Apply thermal paste

3. Which component is usually the first to be installed in a computer system unit?





Correct Answer: c) Processor

4. What should you be cautious of when handling internal components of the computer during disassembly?





Correct Answer: b) Avoiding static electricity

5. Which tool is most commonly used to unscrew parts inside the computer?





Correct Answer: b) Phillips screwdriver

6. What is the purpose of the power supply unit (PSU) during assembly?





Correct Answer: b) To provide power to all the components

7. What is the last step in disassembling a computer system unit?





Correct Answer: d) Disconnect all cables

8. What part connects the CPU to the motherboard?





Correct Answer: b) CPU socket

9. What must be done before installing a new graphics card into the motherboard?





Correct Answer: a) Make sure there is enough space in the case

10. In what order should you generally perform the reassembly of a computer system?





Correct Answer: b) Install processor, RAM, and then connect cables
